Painel de Atualização JFTV

Instruções para Atualização do Site

Este painel foi criado para facilitar a atualização das imagens e informações do site JFTV. Siga as instruções abaixo:

  1. Para atualizar imagens, você precisará substituir os arquivos na pasta images do seu site.
  2. Mantenha os mesmos nomes de arquivos para que as referências no HTML continuem funcionando.
  3. Para filmes e séries, use os nomes filme1.jpg, filme2.jpg, etc.
  4. Para canais, use os nomes canal1.jpg, canal2.jpg, etc.
  5. Para editar textos, você precisará modificar diretamente os arquivos HTML correspondentes.
  6. Após fazer as alterações, salve os arquivos e atualize o site no servidor.

Guia para Edição Manual dos Arquivos

Estrutura de Arquivos

jftv-site/
├── index.html           # Página inicial
├── como-funciona.html   # Página Como Funciona
├── conteudos.html       # Página de Conteúdos
├── ajuda.html           # Página de Ajuda
├── admin.html           # Este painel de administração
├── css/
│   └── style.css        # Estilos do site
├── js/
│   └── main.js          # JavaScript do site
└── images/              # Pasta de imagens
    ├── filme1.jpg       # Imagens de filmes
    ├── filme2.jpg
    ├── ...
    ├── canal1.jpg       # Imagens de canais
    ├── canal2.jpg
    ├── ...
    └── ...

Como Atualizar Manualmente

  1. Para substituir imagens, simplesmente substitua os arquivos na pasta images mantendo os mesmos nomes.
  2. Para editar textos e títulos, abra o arquivo HTML correspondente em um editor de texto.
  3. Localize o texto que deseja alterar e faça as modificações necessárias.
  4. Salve o arquivo e atualize-o no servidor.

Dica:

Para facilitar a localização dos textos nos arquivos HTML, você pode usar a função de busca do seu editor (geralmente Ctrl+F) para encontrar o texto específico que deseja alterar.
